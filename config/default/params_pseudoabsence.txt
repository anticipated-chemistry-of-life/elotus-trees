If one want to change the level of tables generated by the script `prepare_pseudoabsence_tables.R`,
we can change the parameters in the params.yaml file
Default are : structure_taxonomy_npclassifier_03class and organism_taxonomy_05order
(see https://commons-research.github.io/commons-dws-public/notes/2q1wc73tsex1tijg7kzjq3u/ 
for more details.)

structures:
    level:
        structure_taxonomy_npclassifier_01pathway
        structure_taxonomy_npclassifier_02superclass
        structure_taxonomy_npclassifier_03class
        structure_inchikey

organisms:
    level:
        organism_taxonomy_01domain
        organism_taxonomy_02kingdom
        organism_taxonomy_03phylum
        organism_taxonomy_04class
        organism_taxonomy_05order
        organism_taxonomy_06family
        organism_taxonomy_07tribe
        organism_taxonomy_08genus
        organism_taxonomy_09species
        organism_taxonomy_10varietas
